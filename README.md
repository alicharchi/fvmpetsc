# Finite Volume Diffusion Solver (C++ with PETSc)

This project implements  a parallel **1/2D diffusion equation solver** using the **Finite Volume Method (FVM)** in **C++**, with linear system solution handled by **PETSc**.

The numerical algorithm follows the formulation and discretization techniques presented in:

> **H. K. Versteeg and W. Malalasekera**,  
> *An Introduction to Computational Fluid Dynamics: The Finite Volume Method*,  
> 2nd Edition, Pearson Education, 2007.
